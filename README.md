# Shape Analyzer with OpenCV

A computer vision project built with Python and OpenCV that detects and analyzes shapes from user-provided images.

The current implementation focuses on the image processing pipeline required for shape detection. Future versions will include geometric analysis, such as calculating each shape's area, perimeter, and other useful properties.

## Current Features

- Load an image
- Convert the image to grayscale
- Apply Canny edge detection
- Detect contours
- Draw detected contours

## Planned Features

- Calculate shape area
- Calculate shape perimeter
- Identify basic geometric shapes
- Display measurements on the image
